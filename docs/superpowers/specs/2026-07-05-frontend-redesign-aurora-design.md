## Визуальный стиль

### Цветовая схема (из референса)

```css
--bg-primary: #0f0f13;
--bg-secondary: #15151e;
--bg-card: rgba(255,255,255,0.04);
--glass-border: rgba(255,255,255,0.08);
--glass-blur: 12px;
--accent: #E9B1A3;
--accent-hover: #d49a8a;
--accent-secondary: #fbbf24;
--text-primary: #f1f1f3;
--text-secondary: #9b9ba7;
--text-muted: #5c5c6a;
--border: rgba(255,255,255,0.06);
--danger: #ef4444;
--success: #22c55e;
--warning: #f59e0b;
```

### Фон

Aurora — 12-слойный радиальный градиент поверх `--bg-primary`. Слои — сочетания акцентного (peach) и второстепенного (amber) цветов с низкой непрозрачностью.

### Типографика

- **Заголовки/тело:** Onest (300-700)
- **Числа/моно:** DM Mono (400, 500)
- Все тексты контрастные: `--text-primary: #f1f1f3` (почти белый), `--text-secondary: #9b9ba7` (светло-серый, не тёмно-серый)
- `text-shadow: 0 1px 3px rgba(0,0,0,0.45)` для читаемости

### Иконки

Font Awesome 6 Free (через CDN или npm). Заменить текущие эмодзи.

### Компоненты

Glassmorphism-карточки (`backdrop-filter: blur`, полупрозрачные границы). Скругления: `--radius: 16px`, `--radius-sm: 10px`. Кнопки с pill-скруглением (30px).

## Dashboard (`/dashboard`)

### Хедер
- Приветствие + подпись
- Поиск (с иконкой)
- Кнопка «Добавить»

### Баланс (сетка 3 колонки)
1. **Общий баланс** — крупная сумма, изменение за месяц (+2.4%), «обновлено 5 мин назад». Акцентная рамка.
2. **Доходы** — иконка зелёная, сумма, подпись
3. **Расходы** — иконка красная, сумма, подпись

### Две колонки
1. **График** (canvas с динамикой доходов/расходов, переключение Нед/Мес/Год)
2. **Последние операции** — список tx-item с иконкой, названием, описанием, суммой (зелёный/красный)

### Быстрые действия
4 кнопки: Перевод, Доход, Расход, Отчёт
